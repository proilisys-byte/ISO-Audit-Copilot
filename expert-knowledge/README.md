# 🧠 Expert Knowledge — 사용자 노하우 저장소

> 베테랑 심사원의 실전 노하우를 축적하는 지식 저장소입니다.

## 📂 폴더 구조

| 폴더 | 용도 | 예시 |
|------|------|------|
| `must-check-items/` | 심사 시 반드시 확인할 사항 | 규격별 필수 확인 포인트 |
| `common-nonconformities/` | 자주 발생하는 부적합 | 업종별 빈출 NC 패턴 |
| `interview-techniques/` | 심사 인터뷰 기법 | 오프닝/클로징 기법, 질문법 |
| `follow-up-questions/` | 추적 심사 질문 | 후속심사용 핵심 질문 리스트 |
| `industry-risks/` | 업종별 핵심 리스크 | 반도체, 자동차, 식품 등 |

## 📋 사용법

1. 해당 카테고리 폴더에 Markdown(.md) 파일로 노하우를 작성합니다.
2. 파일명은 주제를 명확히 반영하세요.
   - 예: `반도체_웨이퍼팹_필수확인사항.md`
   - 예: `ISO9001_8.5.1_빈출부적합_패턴.md`

## ⚡ AI 응답 우선순위

AI는 답변 생성 시 이 저장소의 노하우를 **2순위**로 참조합니다.
(1순위: Knowledge Base 업로드 자료)
