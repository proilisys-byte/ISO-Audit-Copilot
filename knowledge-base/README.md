# 📚 Knowledge Base — 지식베이스 저장소

> ISO 심사에 필요한 모든 참조 문서를 이 폴더에 저장합니다.

## 📂 폴더 구조

| 폴더 | 용도 | 예시 파일 |
|------|------|----------|
| `iso-standards/` | ISO 규격서 원문 | ISO 9001, ISO 14001, ISO 45001, IATF 16949, AS9100 |
| `checklists/` | 심사 체크리스트 | 규격별/공정별 체크리스트 |
| `audit-cases/` | 심사 사례집 | 부적합 사례, Best Practice |
| `quality-manuals/` | 품질 매뉴얼 | 고객사 품질 매뉴얼 샘플 |
| `procedures/` | 절차서 | 공정별 절차서, SOP |
| `work-standards/` | 작업표준서 | WI (Work Instruction) |
| `fmea/` | FMEA 자료 | Process FMEA, Design FMEA |
| `control-plans/` | Control Plan | 공정별 관리계획서 |
| `spc/` | SPC 자료 | 관리도, 공정능력분석 |
| `msa/` | MSA 자료 | Gage R&R, 측정시스템분석 |
| `8d-reports/` | 8D Report | 고객 클레임 대응 보고서 |
| `capa/` | CAPA 사례 | 시정조치/예방조치 사례 |

## 📋 사용법

1. 해당 카테고리 폴더에 파일을 업로드합니다.
2. 지원 형식: PDF, DOCX, XLSX, PPTX, TXT, MD, Images
3. 파일명에 규격명과 버전을 포함하는 것을 권장합니다.
   - 예: `ISO9001_2015_심사체크리스트_v2.xlsx`

## ⚡ AI 응답 우선순위

AI는 답변 생성 시 이 저장소의 자료를 **최우선(1순위)**으로 참조합니다.
